Face Pixelator

This app detects your face via an experimental face detection API, and then pixelates it. Unfortunately the API isn't stable enough to host on this site, but you can still see the project by:

1. Cloning the repo on Github
2. Going to about://flags on Google Chrome and enabling 'Experimental Web Platform features'
3. Opening index.html
